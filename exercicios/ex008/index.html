Usando a tag <pre> que faz a pré-formatação do texto, mostrando que do jeito que as coisas foram escritas
irão aparecer do mesmo jeito no site.

Exemplo:

    num = int(input('Digite um número: '))
    if num % 2 == 0:
        print (f' O número {num} é PAR')
    else
        print (f' O número {num} é ÍMPAR')
    print('Fim do programa')
